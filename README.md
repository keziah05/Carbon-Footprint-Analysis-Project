# Carbon Footprint Analysis Project

## Overview

This project was completed as part of the 1M1B Green Internship. The objective was to collect, analyze, and interpret environmental data related to carbon emissions and sustainability practices. The project aimed to identify patterns and provide insights that could support environmentally responsible decision-making.

## Objectives

* Collect carbon footprint-related data.
* Analyze environmental trends and emission patterns.
* Visualize data using charts and dashboards.
* Generate insights to promote sustainable practices.

## Tools Used

* Google Forms
* Google Sheets
* Microsoft Excel
* Tableau

## Methodology

Data was collected through structured surveys and organized using spreadsheets. The data was then cleaned, analyzed, and visualized using Tableau. Various charts and dashboards were created to identify trends and present findings in an easy-to-understand format.

## Key Learnings

* Data collection and organization
* Data cleaning and analysis
* Dashboard creation and visualization
* Interpretation of data-driven insights
* Communication of findings through reports and presentations

## Outcome

The project helped identify factors contributing to carbon emissions and highlighted opportunities for adopting more sustainable practices. It also strengthened my analytical, problem-solving, and data visualization skills.
